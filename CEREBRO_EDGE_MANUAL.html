    CEREBRO EDGE 요약 메뉴얼¶

    2026.05.20 기준 Ver 0.1

    1. CEREBRO EDGE¶

    1.1 CEREBRO EDGE란¶

    • 업무 흐름을 시각화하고 실행 구조까지 연결하는 노코드 기반 시스템 구축 플랫폼 입니다.
      방대한 설계 문서를 일괄 입력하면 AI가 이를 분석해 서브시스템 단위의 업무 흐름을 자동 생성하여 시스템 구축 시간을 획기적으로 단축합니다.
      또한 시스템의 소스코드를 자동 분석해 구조, 호출 관계, 데이터 흐름을 파악하여 이를 기반으로 자연어로 시스템을 탐색하고 시스템 구조와 업무 흐름을 쉽게 이해할 수 있습니다.

    1.2 주요 기능¶

    🔀 업무 플로우 디자이너¶

    • 시각적 프로세스 모델링
    • 사용자 친화적 UI 환경
    • 컴포넌트 기반 No-Code 개발
    • 현업-개발자 통합 협업 환경

    🔄 AI 기반 리버스 엔지니어링¶

    • 로직 기반 플로우 자동 변환
    • 비즈니스 규칙(Rule) 식별 및 추출
    • 현행 시스템 자산 분석
    • 코드-플로우 양방향 연계 추적

    🤖 지능형 실행엔진 (LLM 기반)¶

    • LLM 기반 컨텍스트 추론
    • 동적 워크플로우 경로 제어
    • 유연한 모델 및 기능 연계
    • 지능형 데이터 가공 및 결과 도출

    🖥️ 실시간 모니터링¶

    • 엔드투엔드(E2E) 워크플로우 실시간 관제
    • 실시간 시스템 자원 모니터링
    • 장애 추적 및 신속한 원인 분석
    • 상세 오류 로그 추적

    1.3 특장점¶

    ⚡ 빠른 성능¶

    • 대용량 처리 최적화
    • 실시간 응답성 확보
    • 초경량 엔진 아키텍처

    🧩 드래그 앤 드롭¶

    • No-Code 개발 환경 지원
    • 직관적인 최신 UI/UX 제공
    • 웹 기반 비주얼 편집기

    🔀 플로우차트 업무 흐름¶

    • 업무 프로세스 시각화 제어
    • 시스템 가독성 및 관리 효율화
    • 도메인 흐름 가시성 확보

    🛡️ 자동 복구¶

    • Self-Healing 시스템 지원
    • 운영 무중단성 및 안정성 극대화
    • 정상 상태 자동 롤백/재수행

    💰 구축 및 유지보수 비용¶

    • 구축기간 단축
    • 유지보수 비용 절감
    • 비개발자 이해도 상승
    • 업무담당자 개발단계 참여

    2. 로그인¶

    • 이메일 주소, 이름, 전화번호를 입력하여 회원가입을 신청합니다.
    • 가입 완료 후 관리자의 승인을 거쳐 최종 계정이 활성화됩니다.
    • 로그인 완료 시 권한에 따라 메인 화면 또는 비즈니스별 기능이 제어됩니다.

    3. 비즈니스¶

    4. 데이터소스¶

    비즈니스 워크플로우에서 사용할 수 있는 데이터베이스 연결정보를 관리합니다. - 데이터소스를 한 번 등록하면 이후 모든 DB 블록에서 드롭다운으로 선택해 사용할 수 있습니다. - 데이터베이스는 Oracle, MySQL, MariaDB, PostgreSQL 타입을 지원합니다. - 커넥션 풀 크기, 최소 유휴 커넥션 수 등 안정적인 DB 운용을 위한 세부 설정을 관리합니다. - 등록된 데이터소스가 현재 어떤 비즈니스 워크플로우와 DB 블록에서 참조되어 사용 중인지 한눈에 확인하고 추적할 수 있습니다.

    4.1 데이터소스 등록¶

    • DB 블록에서 사용할 데이터베이스 연결 정보를 등록합니다.
    • 데이터소스 추가하기 버튼을 클릭합니다.
    • 데이터소스 이름, 데이터베이스 타입, 호스트, 포트, 데이터베이스 이름, 사용자명, 비밀번호를 입력합니다.
    • 연결 테스트를 통해 정상 접속여부를 확인합니다.
    • 생성 버튼을 클릭하여 데이터소스를 추가합니다.

    5. SQL 쿼리도구¶

    등록된 데이터소스에 직접 SQL 쿼리를 실행하여 결과를 실시간으로 조회하고, 워크플로우 적용 전에 로직을 검증하는 도구입니다. - 시스템 부하를 방지하기 위해 실시간 실행 결과는 최대 500건까지만 제한합니다. - 모든 쿼리 실행 이력을 자동으로 저장하며, 히스토리에서 선택하여 언제든지 재사용할 수 있습니다. - 자주 사용하는 주요 쿼리는 별표 아이콘을 클릭하여 간편하게 즐겨찾기로 등록하고 해제합니다. - 불필요해진 과거 쿼리 이력은 휴지통 아이콘을 클릭하여 히스토리 목록에서 즉시 제거합니다.

    6. 스케줄러¶

    정해진 주기에 따라 특정 비즈니스 워크플로우가 자동으로 실행되도록 예약하고 관리하는 도구입니다. - 이름, 비즈니스, 실행 주기(CronTab)를 설정하여 원하는 시점에 비즈니스를 자동 실행합니다. - 토글 버튼 스위치 조작만으로 스케줄의 사용 여부를 실시간으로 제어합니다.

    7. 로그¶

    비즈니스 워크플로우의 시작부터 완료까지의 모든 처리 이력을 기록하고, 문제 추적 및 데이터 보관을 지원하는 도구입니다. - 워크플로우의 시작과 완료는 물론, 내부의 개별 블록 단위별 수행 시점과 진행 상황을 상세히 기록하고 관리합니다. - 고급 검색 추가 필터를 활용하여 특정 비즈니스의 오류나 데이터 이상 현상을 신속하게 분석합니다. - 원하는 로그를 클릭하여 가독성이 높은 테이블 형태나 표준 JSON 포맷으로 세부 내용을 확인합니다. - 누적된 로그를 압축하여 디스크 공간을 절약하며, 일정에 따른 자동 압축과 즉시 수동 압축을 모두 지원합니다.

    8. 모니터링¶

    전체 비즈니스의 실행 상태와 성능 통계를 한 화면에서 직관적으로 관제하는 통합 대시보드입니다. - 각 비즈니스의 호출 경로(테스트,API,스케줄러 등)와 실행 결과(실행중,완료,실패)를 실시간으로 파악합니다. - 전체 프로세스 총 소요 시간은 물론, 내부 블록별 처리 시간을 제공하여 성능 병목 지점을 확인합니다. - 개별 이력의 구체적인 실행 내용과 데이터 처리 결과를 대시보드에서 즉시 점검합니다.
