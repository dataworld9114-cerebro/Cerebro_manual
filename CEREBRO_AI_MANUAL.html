    1. CEREBRO AI

    1.1 CEREBRO AI란

    CEREBRO AI는 다양한 유형의 데이터를 수집·분석하고, 인공지능 기반 분석 기능을 통해 데이터 간 연관관계와 이상 패턴을 탐지할 수 있는 빅데이터 분석 플랫폼입니다.

    1.2 주요 기능

    📊 데이터 탐색 및 전처리

    • 다양한 데이터 소스로부터 데이터 수집
    • 분석 목적에 맞는 데이터 정제 및 가공
    • 결측치 처리 및 데이터 전처리 기능 제공
    • 분석 가능한 형태의 데이터 구조 변환 지원

    📈 OLAP 분석

    • 다차원 데이터 기반 분석 수행
    • 다양한 기준의 데이터 비교 및 탐색
    • 통계 기반 데이터 패턴 분석
    • 시각화를 통한 인사이트 도출

    🔗 연관관계 분석

    • 데이터 간 연결 관계 시각화
    • 객체 간 관계 및 패턴 분석
    • 이상 패턴 및 연관 구조 탐지
    • 네트워크 기반 관계 분석 지원

    🤖 인공지능 모델링

    • 머신러닝 및 딥러닝 기반 모델 생성
    • 학습 데이터 기반 예측 분석 수행
    • AI 모델 학습 및 결과 예측 지원
    • 분석 자동화 기반 모델 운영 가능

    1.3 특장점

    💬 GPT 기반 대화형 분석 환경

    세레브로 GPT를 활용한 자연어 기반 분석 기능 제공

    • 자연어 질의응답 기반 데이터 분석
    • 분석 결과 요약 및 인사이트 제공
    • 대화형 흐름 기반 사용자 인터페이스 지원

    🧩 원스톱 데이터 분석 환경

    데이터 처리부터 분석까지 통합 환경 제공

    • 데이터 전처리 및 통계 분석 통합 지원
    • 직관적인 사용자 중심 인터페이스 구성
    • 별도 개발 없이 분석 업무 수행 가능

    ⚙️ 인공지능 개발 자동화

    AI 모델 개발 과정을 자동화하여 분석 효율 향상

    • AI 모델 생성 및 학습 자동화
    • 예측 기반 분석 프로세스 지원
    • 반복 분석 업무 간소화

    🛠️ 유연한 확장 구조

    사용 환경에 맞는 기능 확장 및 커스터마이징 지원

    • 요구사항 기반 기능 확장 가능
    • 모듈형 구조 기반 기능 구성
    • 사용자 환경 맞춤형 시스템 적용 지원

    3.1 데이터 선택

    • 분석할 데이터를 선택 및 업로드하는 화면입니다.
      • 상단 분석할 데이터명을 클릭하여 데이터를 확인합니다.
      • 업로드 버튼을 클릭하여 데이터를 업로드 합니다.
      • 확인 버튼을 클릭하여 해당 자료의 원본 정보를 확인합니다.


    3.2 원본

    • 업로드 한 데이터의 원본 자료를 확인하는 화면입니다.
    • 데이터 구조 : 원본 자료에 대한 컬럼별 정보를 조회합니다.
    • 데이터 보기 : 원본 자료에 대한 실제 데이터를 조회합니다.
    • 차트 시각화 : 원본 자료 데이터를 차트를 통해 시각화합니다.

    3.2.1 데이터 구조

    • 원본 자료에 대한 데이터 정보를 조회하는 기능입니다.
    • GPT를 통하여 해당 자료의 컬럼 정보를 요약하여 확인 할 수 있습니다.
    • 자료 내 컬럼별로 데이터 형식, 건수, 중복건, 결측치, 통계정보 등을 파악하여 컬럼 간 문제점을 확인 할 수 있습니다.
    • 자료 컬럼 별 차트를 통해 데이터 분포 현황등을 파악할 수 있습니다.

    3.2.1.1 CerebroGPT 요약
    • CEREBRO AI가 업로드한 데이터 정보를 분석하여 요약본을 볼 수 있는 화면입니다.
    • 상단의 [...] 버튼을 클릭하여 요약 정보를 확인 할 수 있습니다.

    3.2.1.2 컬럼 차트 화면
    • 컬럼 값의 데이터 분포를 차트로 표시합니다
    • 컬럼명 옆 차트 버튼을 클릭하여 데이터 분포를 확인할 수 있습니다.
    • 데이터 컬럼 옆 차트버튼을 클릭하여 컬럼간 데이터를 확인할 수 있습니다.

    3.2.2 데이터 보기

    • 업로드한 원본 데이터를 확인하는 기능입니다.
    • 원본 데이터를 그룹화 및 필터링하여 데이터를 사전에 파악할 수 있습니다.

    3.2.2.1 그룹 색 적용
    • 컬럼별 데이터에 따른 색을 부여하여 화면에 표시합니다.
    • 상단에 그룹색 적용 버튼을 클릭하여 컬럼별 같은 데이터를 확인할 수 있습니다.

    3.2.2.2 데이터 다운로드
    • 조회한 데이터를 csv, xlsx 파일로 변환하여 다운로드합니다.
    • 그룹 색 적용 옆 csv, xlsx 로 저장 버튼을 클릭하여 저장합니다.

    3.2.2.3 너비 간격 조절
    • 화면에 표시된 표의 너비를 변경합니다
    • 오른쪽 바를 드래그 하여 열의 너비를 정렬합니다.

    3.2.2.4 데이터 정렬
    • 컬럼의 데이터를 오름차순 또는 내림차순으로 정렬합니다.
    • 컬럼 옆 정렬 버튼을 통해 데이터를 오름차순/내림차순으로 정렬합니다.

    3.2.2.5 데이터 필터
    • 특정 데이터만 조회하도록 필터링합니다.
    • 컬럼 옆 필터 버튼을 클릭하여 조회할 특정 데이터를 선택한 후 체크버튼을 클릭하여 적용합니다.
    • 쓰레기통 버튼을 클릭하여 필터링을 초기화합니다.

    3.2.3 차트 시각화
    • 원본 데이터에 컬럼별 데이터를 선택하여 시각화하는 기능입니다.
    • 수치형 데이터에 합계/평균, 문자형 데이터에 갯수를 측정값으로 지원합니다.
    • 막대, 선, 히트맵 등 다양한 차트 형식을 지원합니다.

    3.2.3.1 막대 그래프 그리기
    • 막대그래프로 데이터를 시각화 할 수 있습니다.
    • 그래프 선택에서 '막대 그래프' 항목을 선택합니다.
    • 그래프를 그릴 떄 사용할 항목을 검색합니다.
    • 막대그래프에 사용할 분류항목, 비교항목을 지정 후 드래그합니다.
    • 막대그래프에 사용할 대상 값을 지정 후 드래그합니다.
    • 'CREATE' 를 통해 그래프를 생성합니다.
    • 하단 정렬을 통해 값 내림차순으로 정렬합니다.

    3.2.3.2 선 그래프 그리기
    • 선그래프로 데이터를 시각화 할 수 있습니다.
    • 그래프 선택에서 '선 그래프' 항목을 선택합니다.
    • 선그래프에 사용할 분류항목, 비교항목을 지정 후 드래그합니다.
    • 특정컬럼에서 적용에 필요한 값을 선택하여 필터링합니다.
    • 선그래프에 사용할 대상 값을 지정 후 드래그합니다.
    • 'CREATE' 를 통해 그래프를 생성합니다.

    3.2.3.3 레이더 그리기
    • 데이터를 레이더로 시각화 할 수 있습니다.
    • 그래프 선택에서 '레이더' 항목을 선택합니다.
    • 레이더에 사용할 지표 항목을 지정 후 드래그합니다.
      • 지표항목은 숫자만 사용가능합니다.
    • 레이더에 사용할 비교항목을 지정 후 드래그합니다.
    • 'CREATE' 를 통해 그래프를 생성합니다.
    • 하단 정렬을 통해 값 내림차순으로 정렬합니다.

    3.2.3.4 히스토그램 그리기
    • 히스토그램으로 데이터의 분포를 확인 할 수 있습니다.

    • 히스토그램에 사용할 분포항목을 지정 후 드래그합니다.

    3.2.3.5 박스플롯 그리기

    • 그래프 선택에서 '박스플롯' 항목을 선택합니다.
    • 막대그래프에 사용할 측정값,그룹기준 항목을 지정 후 드래그합니다.

    3.2.3.6 산점도 그리기

    • 그래프 선택에서 '산점도' 항목을 선택합니다.
    • 막대그래프에 사용할 가로축,세로축,크기기준, 그룹기준 항목을 지정 후 드래그합니다.
    • 막대그래프에 사용할 대상 값을 지정 후 드래그합니다.

    • 측정값 아래 하단 블록을 클릭한 후 측정할 항목과 측정 값을 지정합니다.

    • 정렬을 통해 오름차순으로 다시 정렬합니다.

    3.2.3.8 영역 그래프 그리기

    • 그래프 선택에서 '영역' 항목을 선택합니다.
    • 막대그래프에 사용할 분류 항목, 비교항목, 측정값 을 지정 후 드래그합니다.
    • 분류 항목에서 특정 값을 기준으로 필터링합니다.
    • 'CREATE' 를 통해 그래프를 생성합니다.
    • 정렬을 통해 오름차순으로 다시 정렬합니다.

    3.3 전처리

    • 전처리 설정 : 선택한 데이터에 전처리를 적용한 뒤 조회 및 분석하기 위한 화면입니다.
    • 데이터 구조 : 전처리 된 자료에 대한 데이터 정보를 조회합니다.
    • 데이터 보기 : 전처리 된 자료에 대한 실제 데이터를 조회합니다.
    • 차트 시각화 : 전처리 된 자료 데이터를 차트를 통해 시각화합니다.

    3.3.1 전처리 설정

    • 사용자 데이터를 다루기 쉬운 자료 형태로 변환하는 화면입니다.
    • 데이터 내 컬럼을 합치거나 가공하여 신규 생성 및 기존 컬럼을 삭제 할 수 있습니다.
    • 데이터 내 결측값, 중복값이 있을 경우 대체하여 가공할 수 있습니다.
    • 이상치 발견 시 정규화 기능을 통해 데이터 스케일링을 지원합니다.

    3.3.1.1 AI전처리
    • GPT가 사용자 데이터를 분석하고, 결측치, 중복값 을 자동으로 전처리 합니다.
      • 결측값처리, 이상치처리, 형변환 을 한 데이터

    3.3.1.2 AI 직접입력
    • GPT에게 전처리를 요청할 내역을 입력합니다.
      • " 사기이면 1 아니면 0 인 컬럼을 만들어줘" 라고 한 결과

    3.3.1.2 필터링
    • 컬럼 내 특정 값만 사용하도록 필터링 합니다.

    3.3.1.3 값 변경
    • 컬럼 내 특정 데이터 값을 변형하여 사용하도록 변경합니다.

    3.3.1.4 형 변환
    • 데이터 내 조건이나 값을 통하여 데이터를 정제합니다.

    3.3.1.5 중복값
    • 데이터 중복 된 값이 있을 경우 제거합니다.

    3.3.1.6 결측값
    • 데이터 내 오류값(#N/A, nan)이 있을 경우 이를 제거합니다.

    3.3.1.7 항목삭제
    • 데이터 내 컬럼을 삭제합니다.

    3.3.1.8 항목명변경
    • 데이터 내 컬럼명을 변경합니다.

    3.3.1.9 항목생성
    • 데이터 내 컬럼을 새로 생성합니다.

    3.3.1.10 정규화
    • 데이터 내 조건이나 값을 통하여 데이터를 정제합니다.

    3.3.1.11 이상치
    • 데이터 내 분포나 범위를 초과한 값을 삭제합니다.

    3.3.2 데이터 구조

    • 전처리 자료에 대한 데이터 정보를 조회하는 기능입니다.
    • GPT를 통하여 해당 자료의 컬럼 정보를 요약하여 확인 할 수 있습니다.
    • 자료 내 컬럼별로 데이터 형식, 건수, 중복건, 결측치, 통계정보 등을 파악하여 컬럼 간 문제점을 확인 할 수 있습니다.
    • 자료 컬럼 별 차트를 통해 데이터 분포 현황등을 파악할 수 있습니다.

      • 우측 상단 필터 버튼을 클릭합니다.
      • 항목별로 필터링할 항목을 선택 후 적용 버튼을 클릭합니다.
      • 선택한 항목별로 노드가 필터링 되어 분포도에 표기됩니다.

    3.4.2.4 검색 필터
    • 항목별 검색 조건에 맞는 노드를 표시합니다.
      • 우측 상단 검색필터 버튼을 클릭합니다.
      • 필터링할 항목을 선택 한 후 + 버튼을 클릭합니다.
      • 검색할 조건을 선택하고 상세 조건을 입력한 후 불러오기 버튼을 클릭합니다.

    3.4.2.5 분석도 레이아웃 변경
    • 분석도의 레이아웃을 변경합니다.
      • 우측 상단 검색필터 버튼을 클릭합니다.
      • 분포도가 가로형, 신경망형, 세로형 순으로 변경됩니다.

    3.4.2.6 설정
    • 분석도에 관하여 노드 간 UI를 설정합니다.
      • 우측 설정 버튼을 클릭합니다.
      • 노드간격(가로, 세로)를 드래그 하여 수정합니다.
      • 이름크기를 드래그 하여 수정합니다.
      • 이름표시 옆 태그를 클릭하여 이름을 숨기거나 보이도록 합니다.
      • 엣지표시 옆 태그를 클릭하여 노드 간 선(엣지)을 숨기거나 보이도록 합니다.
      • 엣저 설정에서 + 를 클릭하여 노드 간 연결된 빈도수 별로 색상을 다르게 표기합니다.
        + 버튼 클릭시 1위부터 순서대로 표기됩니다.

    3.4.2.7 관계
    • 선택한 노드 간 1차 관계를 표시합니다.
      • 우측 관계 버튼을 클릭합니다.
      • 관계를 확인하고자 하는 노드 중 하나를 클릭합니다.
      • 클릭한 노드간 1차 관계를 확인할 수 있습니다.

    3.4.2.8 패턴분석
    • 특정 노드 간 빈도 패턴을 표시합니다.
      • 우측 패턴분석 버튼을 클릭합니다.
